Overview
Build a lightweight web app where users can upload their own data, define what

they want to track, and view it as a personalised dashboard with charts and AI-
powered insights.

Think of it as a stripped-down Mixpanel or Amplitude — built from scratch by you.
This is intentionally open-ended. I'm not evaluating whether you built every
feature. I'm evaluating how you think, the decisions you make, and the quality of
what you ship.

The User Journey
1. User signs up and lands on an empty personal dashboard
2. User uploads a CSV file (e.g. sales data, leads, transactions — any tabular
data)
3. App parses the file, detects column names and data types (number, date, text)
4. User selects which columns to visualise — e.g. "revenue by month" or "leads
by source"
5. Dashboard renders charts based on their selections
6. User can save their dashboard and return to it in a future session
7. User can trigger an AI-generated insight about their data
